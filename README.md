# mos-void

A terminal simulator/tycoon game where Mo is trapped in the void.

## Play (HTML)

Open `/home/runner/work/mos-void/mos-void/index.html` in your browser.

## Play (Terminal)

```bash
python mos_void.py
```

## Build a Windows `.exe`

1. Install PyInstaller:

```bash
python -m pip install pyinstaller
```

2. Build:

```bash
python build_exe.py
```

3. The executable will be generated at:

```text
dist/mos-void.exe
```

## What's new in the detailed edition

- Difficulty selection (Chill / Standard / Nightmare)
- More actions: scout rifts and craft gear from scrap
- Daily mission system with progress rewards
- Expanded progression with drone upgrades and higher beacon cap
- ASCII scene art and richer terminal HUD for a more visual feel

## Goal

Manage Mo's health, energy, mood, resources, and void pressure while buying upgrades and improving the beacon. Reach **100% escape progress** with at least **Beacon Level 3** to escape.
